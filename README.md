### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install libwebp
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> pkg install bash
> npm install -g pm2
> git clone https://github.com/DarrenM999/Miku-Izayoi
> cd Izayoi-Miku
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```

```bash
> npm start
```
###### Run With PM2
```bash
> pm2 start (File Name to Run , Example : main.js or index.js)
> pm2 save
> pm2 monit
```
